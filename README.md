# PMS-система для сети отелей
Приложение для сотрудников отелей, предназначенное для учёта гостей в базе данных. Поддерживает разграничение ролей на Системного администратора, Менеджера и Ресепшиониста.

# Стек
- Frontend: React (Vite + TypeScript)
- Backend: Node.js (Express + TypeScript)
- Database: PostgreSQL

# Адреса при поднятии локально
- Frontend: http://localhost:5173
- API health: http://localhost:4000/api/health

# Важные параметры безопасности
- брутфорс на логин (rate limiting)
- строгая валидация входных данных (Zod)
- RBAC по должности и ограничение по корпусу для сотрудников
- параметризованные SQL-запросы (pg)
